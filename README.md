# SHL Assessment Recommender (Ready-to-run)
This repo contains a ready-to-run implementation of the SHL Assessment Recommendation System
(scraper -> embeddings -> LLM preprocessing -> FAISS retrieval -> API -> Streamlit frontend).

**IMPORTANT:** Two files uploaded by you are included into `data/`:
- `/mnt/data/Gen_AI Dataset.xlsx` (test/train queries)
- `/mnt/data/SHL AI Intern RE Generative AI assignment Updated.pdf` (assignment spec)

See `requirements.txt` and the `scripts/` folder for usage.
